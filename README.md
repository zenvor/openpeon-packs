# openpeon-packs

A curated collection of CESP sound packs for [peon-ping](https://github.com/PeonPing/peon-ping).

## Packs

| Pack | Display Name | Language |
|---|---|---|
| [honor_of_kings](./honor_of_kings) | Honor of Kings | zh |
| [ra2_eva_commander](./ra2_eva_commander) | RA2 EVA Commander | en |

## Usage

Install a pack with peon-ping:

```bash
peon packs use --install honor_of_kings
peon packs use --install ra2_eva_commander
```
